Open Escalar

Scaling with openness through the sky from one cloud to another...


What is OpenEscalar?

It's an open cloud framework that will allow you to deploy, control, orchestrate the different sections of cloud computing starting from IaaS moving into PaaS and finally providing SaaS


Why should I use it?

If you are planning to deploy through multiple cloud vendors, orchestrate using multiple methods (scripts, puppet or chef), create Platform as a Service, contribute to the open source community, then you should give it a try!


How can I contribute?

Build and share.... Tasks... Workflows... Deployments... Infrastructures...




OpenEscalar is an open source supporter.
