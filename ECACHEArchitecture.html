Escalar Cache Services - ECACHE

ECACHE is a highly available memcache service which allows the users to store key-value pairs on vms in order to increase the performance of their applications. Most common use is to sit them between the application servers and the database servers, storing in their memory database results, reducing the overhead of making multiple database calls for the same result. Its technology is based on the opensource Memcached and Linux cluster suite which provides a highly available service.
